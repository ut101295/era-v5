How the Collapse Math Works (Click to toggle) +

For an input vector x (coordinate [x1, x2]), each layer in the 5-layer linear network computes its output directly as:

y1 = x · W1 + b1
y2 = y1 · W2 + b2 = x · (W1W2) + (b1W2 + b2)
y3 = y2 · W3 + b3
...
y5 = y4 · W5 + b5

Substituting each equation into the next expands the final output to:

y5 = x · (W1W2W3W4W5) + (((b1W2 + b2)W3 + b3)W4 + b4)W5 + b5

We define the collapsed variables:

  • Weff = W1 × W2 × W3 × W4 × W5 (size 2 × 1)
  • beff = (((b1W2 + b2)W3 + b3)W4 + b4)W5 + b5 (scalar bias)

This reduces the entire 5-layer network calculation to a single linear layer: y5 = x · Weff + beff. Without non-linear activations (like ReLU) in between, the depth collapse is complete, and the network is mathematically restricted to drawing a straight line boundary.
