Roundtrip gate — faithfulness

The gate requires decode(encode(text)) to preserve every non-whitespace character. Byte-level submissions failed this by dropping [ / ]; this one passes because Metaspace keeps punctuation attached, NFC is non-lossy, and byte_fallback covers anything unseen.

✓ byte_fallback is functional (256 <0xHH> tokens) — any character round-trips via its UTF-8 bytes, so nothing is ever silently dropped.

Why this design passes the faithful-roundtrip gate

NFC normalization is canonical and reversible — unlike NFKC it never rewrites characters (e.g. curly → straight quotes), so no visible symbol is lost. Metaspace turns the space U+0020 into a visible marker ▁ and keeps brackets, URLs, apostrophes and number separators attached to their word, tokenizing on characters (not 3-byte UTF-8 sequences) so Indic scripts compress well. byte_fallback is enabled so any symbol never seen in training still round-trips through its UTF-8 bytes rather than being dropped. Decode reverses it all: Sequence(ByteFallback, Metaspace) rebuilds bytes and turns every ▁ back into a space. Fertility is tokens / faithful_units per language; the score is 1000 / (max_fertility − min_fertility), and the Hindi penalty exp(max(0, hi/1.2 − 1)) is 1.0 because Hindi fertility sits well under 1.2.
